## Markdown Rendering

This project now includes a MarkdownRenderer component for rendering markdown content to HTML. To use it in your components:

1. Import the component:
   ```javascript
   import MarkdownRenderer from './src/components/MarkdownRenderer.vue'
   ```

2. Use it in your template:
   ```vue
   &lt;MarkdownRenderer :markdownContent="yourMarkdownString" /&gt;
   ```

The component uses the Marked.js library for parsing markdown and applies basic styling to the rendered HTML elements.
